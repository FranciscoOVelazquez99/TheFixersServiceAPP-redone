Para acceder a la aplicación, abra su navegador e ingrese la siguiente URL: http://<IP_DE_ESTA_PC>:6969 o desde el icono de la aplicación.


==Credenciales de acceso inicial:==
    El usuario y la contraseña predeterminados son: admin / admin. Puede modificar esta contraseña desde la sección de administración de usuarios para mejorar la seguridad del sistema.

==Requisitos de configuración para el correcto funcionamiento del servicio:==
    Instalar MySQL. [root localhost 3306]
    Disponibilidad del Puerto 6969: Asegúrese de que el puerto 6969 esté libre y no en uso por otras aplicaciones para permitir el acceso a la aplicación.
    Acceso a través del Firewall: Para que otros usuarios puedan conectarse, configure el firewall para permitir el tráfico en el puerto 6969.

==Acceso de otros usuarios:==
    Los usuarios adicionales pueden ingresar a la aplicación utilizando la URL http://<IP_DE_ESTA_PC>:6969 siempre que cumplan con los requisitos de acceso mencionados.

    Esta configuración garantiza el funcionamiento y la accesibilidad de la aplicación en la red.
